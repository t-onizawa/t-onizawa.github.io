VPSでの準備

VPS 基本 Ruby on Rails 環境構築 CentOS

投稿日2015/01/20

VPSでの準備を説明していきます。VPSはさくらのVPSを使います。OSはCentOSです。環境構築を何回かに分けて明記します。

VPSの起動

https://secure.sakura.ad.jp/vps/にログインしてサーバーを起動します。

sshで接続

作業マシン

ssh root@契約したIPアドレス
  • パスワード（契約した時の管理用初期パスワード）を入力します。

※パスワード入力前に(yes/no)を聞かれるのでyesを入力。

Vimをインストール

作業マシン

yum -y install sudo vim

一般ユーザの作成

useradd -m yourname
passwd yourname
  • yournameの部分は適宜変更してください。
  • パスワードを聞かれるので2回入力。

sudo権限の付与

ユーザyournameにsudo権限を付与します。

vi /etc/sudoers.d/00_wheel

で新規ファイルを作成し、内容を

%wheel ALL=(ALL) ALL

で作成し、下記コマンドを入力

chmod 400 /etc/sudoers.d/00_wheel
gpasswd -a yourname wheel
