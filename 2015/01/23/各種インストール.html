各種パッケージのインストールについて説明していきます。インストール、初期化を行います。環境構築を何回かに分けて明記しています。OSはCentOSです。

各種インストール

SSHでログインします。

ssh yourname@契約したIPアドレス

インストールします。

sudo yum -y install gcc gcc-c++ make autoconf openssl-devel readline-devel libyaml-devel mysql-devel wget git

Rubyのビルド

rbenvをシステムワイドにインストール

ユーザをrootに変更

su
cd /usr/local

クローン

git clone https://github.com/sstephenson/rbenv.git ./rbenv

環境設定

新規ファイル

vi /etc/profile.d/env.sh

下記を記載

export RBENV_ROOT="/usr/local/rbenv"
PATH="$RBENV_ROOT/bin:$PATH"
export PATH
eval "$(rbenv init -)"

ruby-build を インストール

cd /usr/local/rbenv
mkdir ./plugins
cd ./plugins
git clone https://github.com/sstephenson/ruby-build.git ./ruby-build

Rubyをインストール

お使いのバーションをインストールしてください。

rbenv install -v 2.0.0-p353

再読み込み

rbenv rehash

設定

rbenv global 2.0.0-p353

確認

ruby -v

RubyGems

更新

gem update --system

確認

gem --version

Bundler

インストール

gem install bundler --no-ri --no-rdoc

確認

bundle -v
